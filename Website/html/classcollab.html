Class Collab

Github

Every classroom is unique! If you put together the type of minds involved and the variety of knowledge passed in a classroom, you'll realize that no two classrooms are similar to each other. Class Collab epitomizes this notion!

Class Collab provides a stack overflow-esque platform where the students of a classroom can create/join a channel, ask questions and share their learnings!

Create a classroom channel!

Join a classroom channel!

Once created/joined a class channel, you can navigate to all of them from the home screen. You are all good to start asking question or sharing your knowledge!
